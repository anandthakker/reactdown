Usage: reactdown [-ohvs] [filename]

Compile reactdown into React component.

Options:

       -o, --output file   Specify output.

  -c, --component module   Module id which will be used as a component.

      -s, --scope module   Module id which will be used as a scope.

              -h, --help   Show this message and exit.

           -v, --version   Show reactdown version and exit.
